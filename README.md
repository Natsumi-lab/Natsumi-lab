## 💻 Portfolio

### 🥜 Nuts Balance  
ナッツ摂取を記録・可視化する習慣化アプリ

**使用技術**  
Next.js / TypeScript / Supabase / PostgreSQL / Tailwind CSS

**主な機能**
- ナッツ摂取記録（チェック形式）
- 栄養バランスの可視化
- 月次レポート
- 連続記録日数（ストリーク）表示
- ダークモード対応

🔗 Repository  
https://github.com/Natsumi-lab/nuts-balance
